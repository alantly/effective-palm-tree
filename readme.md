# Effective Palm Tree

[Postman](https://www.getpostman.com/collections/5e17fef838683470e176)

## build
```
yarn
yarn run build
```

## clean
```
yarn run clean
```

## Run server
```
yarn run build
yarn start
```

## Run tests
```
yarn run build
yarn test
```

## Run dev server
In two terminals run `yarn run bw` and `yarn run watch`
